## 一、架构定义

```
L-1 信任根    宪法本文·Git版本化·不可篡改       强制

L1  全文      SQLite FTS5·BM25·独立部署每节点      强制
L2  基因      LGE JSONL·语义记忆·provenance溯源    强制

```

## 四、格式铁律

```
L-1 Git:      纯文本·UTF-8·.gitattributes
L0 Markdown:  RFC 7763·无私有扩展
L1 SQLite:    FTS5·公有领域·SQL标准
L2 JSONL:     每行合法JSON·带provenance·gzip压缩
L3 文档:      原始文件+Markdown索引
L4 Neo4j:     Cypher导出·CSV边表·可选
L5 桥:        JSON over HTTP·无状态
L6 会话:      JSONL·每行一条消息

  $ pkg install python3 sqlite3 git
  $ git clone lgox-memory-root
  $ python3 restore.py --all

```
