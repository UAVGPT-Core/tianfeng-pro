# 任务：激活百度智能点播VOD视频生产平台

## 背景
千帆VOD(DeepSeek)已榨干。智能点播VOD(视频生产)控制台可进但API未激活。

## 执行步骤

### 1. 打开VOD控制台
浏览器 → https://console.bce.baidu.com/vod2/

### 2. 找「开通/激活」按钮
首页或概览页，找「立即开通」「激活服务」「免费试用」任一字样的按钮，点击。

如果没有明显按钮：
- 左侧菜单 → 概览 → 看是否有开通引导
- 左侧菜单 → 媒资管理 → 点进去看是否触发开通
- 看页面顶部是否有黄色提示条「服务未激活」

### 3. 如果已开通
看左侧菜单是否全部可点（媒资管理、音视频管理、视频生产等）。如果都能点进去显示空列表（非404/无权限），说明已开通——告诉灵龙当前状态。

### 4. 开通后验证
在灵龙终端运行(开通后告诉灵龙，灵龙会自测):
```
python3 ~/lgox-ops/scripts/vod_pro_client.py --test-media
```
