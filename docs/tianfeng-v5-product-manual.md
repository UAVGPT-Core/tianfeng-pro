# 天锋PRO v5.0 — 产品说明书

## 产品概述

天锋PRO是LGOX联邦的首款旗舰产品——世界级AI编程工具。基于2035永动架构，融合9大引擎，实现基因驱动的代码生成、三重智能审查和联邦协同编程。

## 产品定位

- **品牌**: 天锋PRO (Tianfeng PRO)
- **版本**: v5.0.0
- **定位**: AI编程工具
- **Slogan**: 基因驱动的世界级AI编程工具
- **对标**: Cursor($600亿估值)、Codex CLI(97K⭐)、Qoder CN
- **差异化**: 基因记忆×自我进化×联邦共享×零成本×9引擎融合

## 技术架构

```
天锋PRO v5.0 · 九引擎流水线

感知层: 外部雷达(cron每6h) → 行情管道(每1min)
知识层: KPS光合(每4h) → GPC进化(每30min) → FTS5+LGE双检索
生成层: 基因注入(每3h) → 代码大脑(每3min自弈) → MLGE多语言
审查层: AST重写(每2h) → 五角思辨(每6h)
闭环层: FCL总协调(每60min) → DPS推送(每30min)
联邦层: FGI基因互联网(每2h) → GRP/1.0复制协议
```

## 安装部署

### 方式一: pip安装(推荐)

```bash
pip install tianfeng-pro==5.0.0
tianfeng version
```

### 方式二: 一键脚本

```bash
curl -sL https://stock.uavgpt.com/scripts/tianfeng-install.sh | bash
```

### 方式三: 源码安装

```bash
git clone https://github.com/lgox/tianfeng-pro.git
cd tianfeng-pro
pip install -e .
```

### 系统要求

- Python 3.9+
- macOS / Linux / Windows(WSL2)
- 网络: 可选(离线模式可用本地功能)
- 磁盘: ~50MB(核心) + ~200MB(基因库·可选)
